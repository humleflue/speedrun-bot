**Here's the list of available commands:**
`add *game* *time*` // Adds a speedrun record for you
`get *game*` // Gets the leaderboard for the game
`get *game* *player*` // Gets the best times for the player **mentioned**
`get *game* *role*` // Gets the best times within the role **mentioned** (use with caution - use the command below instead if possible)
`get *game* $*roleID*` // Gets the best times within the role with the given ID (remember to mark the id with a '$' as prefix)

`*game*`: Is the name of the game without spaces
`*time*`: Is the time of the finished run (separated by colons eg. hh:mm:ss:ms)
`*player*`: Is the name of the player (The players discord username)
